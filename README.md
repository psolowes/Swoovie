# Swoovie

Next.js 14 + TypeScript starter with Prisma (SQLite), Vitest, ESLint, and Prettier.

## Setup

```bash
npm install
```

## Database

```bash
npm run db:setup
```

## Development

```bash
npm run dev
```

## Testing

```bash
npm run test
```

## Linting & Formatting

```bash
npm run lint
npm run format
```
